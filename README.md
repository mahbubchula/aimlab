# AIM Lab - Artificial Intelligence in Mobility

A premium academic research lab website showcasing AI research in transportation and mobility.

## About

AIM Lab (Artificial Intelligence in Mobility) is dedicated to advancing transportation through artificial intelligence research. Our work spans autonomous vehicles, smart infrastructure, and traffic optimization.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling with dark mode AI aesthetic
- **Framer Motion** for smooth animations

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Build for Production

```bash
npm run build
npm start
```
